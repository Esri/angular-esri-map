Binding Properties Between AngularJS and Esri

Developers can bind properties between Esri JSAPI 4.x and Angular 1.x by making use of watching JSAPI properties as well as Angular's $scope.$applyAsync() or $scope.$apply().

But, as an example, if a user input element only needs to make changes to Esri JSAPI properties, then you can use ng-model to achieve simple "1-direction" binding from Angular to JSAPI.

The table below can be used as a guide to help you decide which methods to use when binding properties between the two frameworks.
