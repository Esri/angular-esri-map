esriLoader
service in module esri.core

Description

Use esriLoader to lazy load the ArcGIS API for JavaScript or to require individual API modules.

Dependencies

  • $q

Methods

  • bootstrap(options)

    Loads the ArcGIS API for JavaScript.

    Parameters
    ParamTypeDetails
    options
    (optional)
    		Object

    Send a list of options of how to load the ArcGIS API for JavaScript. This defaults to {url: 'https://js.arcgis.com/4.4'}.

    Returns
    Promise

    Returns a $q style promise which is resolved once the ArcGIS API for JavaScript has been loaded.

  • isLoaded()

    Returns
    Boolean

    Returns a boolean if the ArcGIS API for JavaScript is already loaded.

  • require(modules, callback)

    Load an Esri module(s) using the Dojo AMD loader.

    Parameters
    ParamTypeDetails
    modulesArrayString

    An array of module strings (or a string of a single module) to be loaded.

    callback
    (optional)
    		Function

    An optional function used to support AMD style loading.

    Returns
    Promise

    Returns a $q style promise which is resolved once modules are loaded.
