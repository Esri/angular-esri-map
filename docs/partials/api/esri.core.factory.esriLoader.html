esriLoader
service in module esri.core

Description

Use esriLoader to lazyload the Esri ArcGIS API or to require API modules.

Dependencies

  • $q

Methods

  • bootstrap(options)

    Loads the Esri ArcGIS API for JavaScript.

    Parameters
    ParamTypeDetails
    options
    (optional)
    		Object

    Send a list of options of how to load the Esri ArcGIS API for JavaScript. Defaults to {url: '//js.arcgis.com/4.0beta3'}

    Returns
    Promise

    Returns a $q style promise which is resolved once the ArcGIS API for JavaScript has been loaded.

  • isLoaded()

    Returns
    Boolean

    Returns a boolean if the Esri ArcGIS API for JavaScript is already loaded.

  • require(modules, callback)

    Load an Esri module using the Dojo AMD loader.

    Parameters
    ParamTypeDetails
    modulesStringArray

    A string of a module or an array of modules to be loaded.

    callback
    (optional)
    		Function

    An optional function used to support AMD style loading, promise and callback are both added to the event loop, possible race condition.

    Returns
    Promise

    Returns a $q style promise which is resolved once modules are loaded
